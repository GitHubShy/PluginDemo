# PluginDemo
Andoird插件调研
-----------------------------------

    加载sd卡上的最简单view（`没有应用res资源的，应用到res资源的在下一版加上`）和apk。类似从服务器下载插件（view和apk）到sdcard然后加
    载view或apk。完美的模板应该是于类似qq游戏和hotfix。
